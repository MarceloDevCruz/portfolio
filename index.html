Cursos & Estudos

JavaScript e TypeScript

Curso voltado tanto para utilizar recursos nativos do JavaScript Vanilla JS quanto na utilização de frameworks e bibliotecas adicionais, trabalhando com o Node.JS e Express na parte do back-end e também no navegador front-end, Utilizando Babel e Webpack para sistema compatibilidade de módulos do ES6 em navegadores mais antigos. O cursos também consiste na aprendizagem da framework React, onde se é utilizados React Hooks, Redux + e Redux Saga, além de ter sessões de Banco de dados SQL com MariaDB e noSQL com MongoDB, no final criaremos uma API Rest, atráves do modelo MVC de arquitetura de software seguindo os padrões de Design Patters, e Orientação a objetos.

Desenvolvimento Web Completo

O curso Desenvolvimento Web Completo conta com mais de 570 aulas, ao todo são mais de 110 horas de videoaulas em que são abordadas as principais tecnologias web do momento, o pacote reúne 20 cursos, entre as tecnologias: HTML5, CSS3, BootStrap 4, Design responsivo Java Script, ES6 (JavaScript Moderno), PHP7, Orientação a Objetos, MySQL, PHP com PDO, Ajax, JQuery, MVC, APIs, IONIC, WordPress, SASS.

Além dessas linguagens de back-end também cursei desenvolvimento web onde aprendemos html, css e javascript onde tivemos que construir um site para web.

React

Curso de react voltado para iniciantes na biblioteca do react, onde explora do mais básico até o o mais avançado do react. Tópicos separados por, Fundamentos do react, componentização, CSS com react, Formulários, com hooks de tratamento de eventos, validação e envio para API, requisições http, React router, context API, e redux. Além dos pontos citados o curso apresenta imersão do full stack com tecnologias criada atráves dos projetos do curso, como o Node.js, Express, MongoDB com Mongoose Firebase, além disso, o curso propoem módulos extras como Criação de APIs e TypeScript com react.
